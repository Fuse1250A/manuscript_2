Интернет магазин книг. Можно переделать для любых товаров, не требующих больших подробных фото (например радиодеталей)
С помощью админки (add2cat.php) через 4 формы закидуются в базу данных товары.
1. Название (автор):  форма принимает только строковые значения (string) напр. Зеленая миля (Стивен Кинг)
2. Фото (имя файла из папки img_product без .jpg): форма принимает только строковые значения (string) напр. zelmil
   форма принимает только строковые значения (string)
   сюда вписывается имя файла (из папки manuscript/img/img_product) без .jpg
   Например. Фоткаете товар, уменьшаете фото, до 200 пикселей в ширину (чтобы не занимало много места
   на хостинге) переименовываете фото в название, состоящее, только из букв (например zelmil.jpg)
   и в данное поле вписываем zelmil
3. Год издания: форма принимает только числовые значения (int) напр. 2012
4. Цена: числовые значения (int) напр. 175

После того, как данные попали в базу, товар отображается на главной странице catalog.php в виде таблицы
Работает добавление товара в корзину, удаление из корзины, оформление заказа (пишется в лог файл) также
сделанные заказы можно просмотреть через админку (manuscript/admin/orders.php)
На странице отзывов работает добавление отзывов и удаление отзывов

Писалось на процедурном php 7.1
mysql 5.5
Сделал дампы двух баз: 
gbook.sql(база данных отзывов)
eshop.sql(база данных каталога магазина)
